## StatusCake 是什么

StatusCake 是一个网站监控平台，说白了就是帮你盯着网站有没有掉线、速度够不够快。从2012年到现在，已经服务了超过12万用户，Netflix、IBM、UNICEF 这些大公司都在用。

这工具最厉害的地方在于，它可以每30秒就检查一次你的网站，从全球30个国家、43个监测点同时进行测试。这意味着，如果你的网站在某个地区访问不了，你马上就能知道。

核心功能包括：

**在线状态监控**：支持 HTTP、HEAD、TCP、DNS、SMTP、SSH、PING、PUSH 共8种测试类型，可以从不同角度检查网站可用性。

**页面速度监测**：分析网站加载速度，找出拖慢速度的元素。这对 SEO 很重要，Google 明确表示页面速度是排名因素。

**SSL 证书监控**：在证书到期前30天、7天、1天分别提醒你，避免因证书过期导致网站显示不安全。

**域名监控**：追踪域名续费日期，防止忘记续费导致域名被抢注。

**服务器监控**：监测 CPU、RAM、磁盘使用情况，超过设定阈值就发警报。

**团队协作**：可以创建联系人组，一旦出问题，整个团队同时收到通知。
